분야별로 해당되는 업무를 분류해 업무수행 방법을 찾을 수 있음

실제 업무수행방법을 작성해 두어 빠른 업무 적응이 가능

연도별 업무수행방법을 전부 작성해 두어 언제나 최신 업무수행방법을 확인 가능

주의사항

그럴 경우 해당 내용을 게시판에 작성하면, 검토 후 반영(미구현)
